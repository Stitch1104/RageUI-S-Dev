# RageUI-S-Dev

# Installation
  
  Pour l'installer ces tres simple vous le metter directement dans votre script et vous ajouter dans le fxmanifest

    client_scripts {
      "src/RMenu.lua",
      "src/menu/RageUI.lua",
      "src/menu/Menu.lua",
      "src/menu/MenuController.lua",
      "src/components/*.lua",
      "src/menu/elements/*.lua",
      "src/menu/items/*.lua",
      "src/menu/panels/*.lua",
      "src/menu/windows/*.lua"
    }
